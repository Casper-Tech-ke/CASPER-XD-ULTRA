# Test Project

## Overview
A simple Node.js/Express web application serving static files.

## Project Architecture
- `index.js` - Express server entry point, serves static files from `public/`, binds to port 5000
- `public/` - Static assets directory
  - `index.html` - Main HTML page
- `package.json` - Node.js project config with Express dependency

## Running
The app starts with `node index.js` and serves on port 5000.
